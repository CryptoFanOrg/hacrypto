typedef fb_t fb2_t[2]

Represents a quadratic extension binary field element.

Definition at line 52 of file relic_fbx.h.

typedef fb_t fb4_t[4]

Represents a quartic extension binary field element.

This extension field is constructed with the basis {1, s, t, st}, where s is a quadratic non-residue in the binary field and t is a quadratic non-residue in the quadratic extension field (s^2 = s + 1 and t^2 = t + 1).
