Typedef Documentation

typedef align dig_t fp_t[FP_DIGS+PADDING(FP_BYTES)/(FP_DIGIT/8)]

Constant used to indicate that there's some room left in the storage of prime field elements. This can be used to avoid carries. Represents a prime field element.

A field element is represented as a digit vector. These digits are organized in little-endian format, that is, the least significant digits are stored in the first positions of the vector.
