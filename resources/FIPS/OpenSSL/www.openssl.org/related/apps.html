Applications

These are references to various Open Source applications that are using the old SSLeay package or the new OpenSSL package.
  • Apache-SSL
    http://www.apache-ssl.org/
    The SSL integration for Apache by Ben Laurie
  • Apache mod_ssl
    http://www.modssl.org/
    The SSL integration for Apache by Ralf S. Engelschall
  • pkcs12
    http://www.drh-consultancy.demon.co.uk/pkcs12faq.html
    A PKCS#12 format utility by Stephen Henson
  • ca-fix
    http://www.drh-consultancy.demon.co.uk/ca-fix.html
    A X.509 certificate patching utility by Stephen Henson
  • Net::SSLeay
    http://search.cpan.org/perldoc?Net::SSLeay
    The Perl module interfacing to SSLeay/OpenSSL functions
  • cURL
    http://curl.haxx.se/
    A sophisticated FTP, HTTP, HTTPS, GOPHER and DICT gathering tool
  • stunnel
    http://stunnel.mirt.net/
    An SSL encryption wrapper for standard network daemons
  • stunnel-p
    http://www.stunnel.org/
    A new branch of the SSL encryption wrapper for standard network daemons
  • sslwrap
    http://www.rickk.com/sslwrap/
    Another SSL encryption wrapper for standard network daemons
  • BSAFEeay
    https://www.cypherpunks.to/bsafeeay/

  • RaidenFTPD for Windows
    http://www.raidenftpd.com/en/

  • GLOBUS
    http://www.globus.org/

  • MICO
    http://www.mico.org/

  • Samba
    http://www.samba.org/

  • Roxen Challenger
    http://www.roxen.com/

  • Kermit
    http://www.kermit-project.org/
    The popular Kermit suite of communication programs with exp. SSL/TLS support
  • OpenCA
    http://www.openca.org/
    A proposed Certificate Authority management software based on OpenSSL
  • pyCA
    http://www.pyca.de/
    Certificate Authority package implemented with Phyton and OpenSSL
  • Sendmail
    http://www.sendmail.org/

  • SafeGossip

  • XMLSec

  • DelphiImport
